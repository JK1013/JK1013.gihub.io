安装nodejs

第一步： 到nodejs官网下载Linux的nodejs压缩包到桌面
第二步： 启动Linux，使用shell命令 rz 打开文件上传
第三步： 解压nodejs压缩包 tar -xzvf xxxxxx.gz
第四步： 删除压缩包 rm -rf xxxxxx.gz
第五步： 配置环境变量：
1. export NODEJS_HOME=/root/node-v12.14.0-linux-x64
2. export PATH=$PATH:$NODEJS_HOME/bin
第六步： 配置淘宝镜像 npm install -g cnpm –refistry=https://registry.npm.taobao.org
第七步：创建文件夹 mkdir hexo
第八步：下载客户端 cnpm install -g hexo-cli
第九步：初始化博客 hexo init
